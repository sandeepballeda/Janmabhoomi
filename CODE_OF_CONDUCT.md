# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, inclusive, and harassment-free for everyone.

## Expected Behavior

- Use welcoming and respectful language.
- Accept constructive feedback gracefully.
- Focus on what is best for users and maintainers.
- Be mindful that civic-service software can affect vulnerable communities.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Dismissing safety, privacy, or accessibility concerns.
- Sharing real citizen data in public project spaces.

## Enforcement

Maintainers may remove comments, reject contributions, or restrict participation when behavior violates this code. Serious concerns should be reported privately to the maintainers.
